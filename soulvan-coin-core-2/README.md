# soulvan-coin-core
Soulvan Coin: A Scalable, Secure, and Decentralized Future for Digital Currency

Soulvan Coin is a next-generation cryptocurrency built on the foundational principles of decentralization, peer-to-peer exchange, and open participation. Inspired by the Bitcoin protocol, Soulvan Coin aims to provide a robust platform for digital transactions that prioritizes security and scalability.

## Project Structure

- **src/**: Contains the source code for the application.
  - **main.ts**: The main entry point of the application, initializing the core functionality of Soulvan Coin.
  - **types/**: Contains TypeScript types and interfaces for type safety.
    - **index.ts**: Exports various types used throughout the application.
  
- **build/**: Contains build configurations for different operating systems.
  - **windows/**: Contains Windows-specific build configurations.
    - **build-config.json**: Configuration settings for building the application on Windows.

- **package.json**: Lists the project's dependencies, scripts, and metadata for npm.

- **tsconfig.json**: Configuration file for TypeScript, specifying compiler options and files to include.

## Building the Application

To build the Soulvan Coin application for Windows, navigate to the `build/windows` directory and use the provided `build-config.json` to configure your build environment. Ensure that all platform-specific dependencies are installed.

## Running the Application

After building the application, you can run it using the appropriate command for your operating system. Refer to the documentation in the `package.json` for available scripts.
