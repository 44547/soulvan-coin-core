# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2025-11-04

### Added
- Multi-platform builds (Windows, Linux, macOS) with downloadable zips per platform
- Unity CI via game-ci with artifact caching and centralized release packaging
- GitHub Actions workflow for automated builds and releases
- Direct download links and platform guidance in README
- RELEASE_NOTES.md template for release documentation

### Changed
- Artifact names standardized to SoulvanCore_<platform>.zip
- README updated with multi-platform download information

### Breaking Changes
- Standardized artifact naming convention
